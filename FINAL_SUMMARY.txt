   • Full conversation history available

   • Click "Scegli cartella"
   • Select new folder
   • New folder is remembered

SYSTEM ARCHITECTURE
================================================================================

DATA FLOW:

  User Session
       ↓
  Streamlit App
       ↓
  Session Persistence ←→ File Storage (JSON)
       ↓
  Topic Analyzer ←→ Topic Cache
       ↓
  Topic UI Renderer
       ↓
  Browser Display

PERSISTENCE FLOW:

  App Startup
    ↓
  Load Session State ← current_session.json
    ↓
  Load Cached Documents ← documents_cache.json
    ↓
  Get Saved Folder ← user_settings.json
    ↓
  Load Topics ← topics_cache.json
    ↓
  App Ready (Instant)

STATE SAVING:

  User Action
    ↓
  Process Input
    ↓
  Update Session State
    ↓
  Save Session State ← Save to JSON
    ↓
  Display Result
