HOW TO USE
================================================================================

STARTING THE APPLICATION:

1. Run the startup script:
   .\restart_app.bat

2. Or manual command:
   python -m streamlit run app_streamlit_real_docs.py --server.port 8503

3. Browser automatically opens:
   http://localhost:8503

FIRST TIME SETUP:

1. Click "Scegli cartella" button
2. Select folder with your documents
3. Wait 5-15 seconds for indexing
4. Documents appear and are cached
5. Topics automatically extracted

SUBSEQUENT SESSIONS:

1. Run: .\restart_app.bat
2. Folder automatically selected
3. Documents instantly loaded from cache
4. Topics immediately available
5. Conversation history restored
6. Start using immediately

================================================================================
DATA PERSISTENCE
================================================================================

WHAT GETS SAVED:

1. Document Folder Path
   Storage: data/session_persistence/user_settings.json
   Restored: On every app start
   Duration: Until changed

2. Loaded Documents
   Storage: data/session_persistence/documents_cache.json
   Size: ~1-50 MB depending on documents
   Restored: On app start (instant load)

3. Conversation History
   Storage: data/session_persistence/current_session.json
   Contains: All queries, responses, interactions
   Restored: On app start

4. Topic Analysis
   Storage: data/topics_cache/topics_cache.json
   Contains: Extracted topics, groupings, statistics
   Restored: Instantly on app start

5. Application Settings
   Storage: data/session_persistence/user_settings.json
   Contains: User preferences, folder path
   Restored: On app start

STORAGE LOCATION:
  Local machine only
  Path: ./data/session_persistence/ and ./data/topics_cache/
  No cloud upload
  Full user control

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: App won't start
SOLUTION:
  1. Check port 8503 is free
  2. Use restart_app.bat (handles automatically)
  3. Or: taskkill /f /pid <PID>

ISSUE: Folder not remembered
SOLUTION:
  1. Delete: data/session_persistence/user_settings.json
  2. Restart app
  3. Select folder again

ISSUE: Cache seems broken
SOLUTION:
  1. Delete: data/session_persistence/documents_cache.json
  2. Restart app
  3. Select folder to re-cache

ISSUE: Slow on first load
SOLUTION:
  This is normal! First load indexes documents.
  Subsequent loads will be instant from cache.
