[TROUBLESHOOTING]

Q: "Port 8501 already in use"
A: Kill previous process: taskkill /F /IM python.exe
   Or use different port: streamlit run app.py --server.port 8502

Q: "Path does not exist"
A: Run helper: python find_documents_folder.py
   Verify path with: dir "C:\path\to\documents"

Q: "No documents found"
A: Check folder has PDF, TXT, or MD files
   Make sure files are directly in folder (not in subfolders)

Q: "ModuleNotFoundError"
A: Install dependencies: pip install -r requirements.txt

Q: "Gemini API Error"
A: Verify API key: cat .env | grep GEMINI_API_KEY
   Ensure internet connection

Q: "Embedding taking too long"
A: Normal for 50+ PDFs (5-10 minutes)
   Try with fewer docs first (10-20)
